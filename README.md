# uniapp-template

### Project setup
```
npm install
```

### Project structure

``` bash
├── api                                     // 存放项目api
├── common                                  // 公共函数、样式
├── components                              // 组件
├── libs                                    // 项目用到的插件
├── mixins                                  // 存放全局mixin
├── pages                                   // 项目页面
│   ├── main-pack                           // 主包
│   ├── sub-pack-a                          // 分包A
├── utils                                   // 项目方法class
│   └── request.js                          // uni.request封装
├── static                                  // 公共资源
│   ├── images                              // 本项目相关的图片 
├── store                                   // vuex
│   ├── modules                             // vuex模块分类
│   └── index.js                            // vuex方法集合
├── unpackage                               // 项目编译后的文件
├── .eslintrc                               // eslint规则配置
├── App.vue                                 // 项目主界面
├── main.js                                 // 程序入口文件，加载各种公共组件
├── manifest.json                           // uni-app项目类型及环境配置
├── pages.json                              // 项目路由及项目界面配置
├── README.md                               // 项目介绍文件
└── uni.scss                                // uni-app内置的常用样式变量
```
